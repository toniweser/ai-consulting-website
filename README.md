# KI-Beratung Website

Website for [ki-workshop-unternehmen.de](https://ki-workshop-unternehmen.de) — AI consulting by Christian Braun & Tobias Weber. We help developer teams and mid-sized companies integrate AI into their workflows.

## Tech Stack

- **Framework:** Astro v6 (Static Site Generation)
- **Styling:** Tailwind CSS v4
- **Font:** Space Grotesk (self-hosted via @fontsource)
- **Hosting:** GitHub Pages with GitHub Actions CI/CD

## Development

```sh
npm install
npm run dev       # Dev server at localhost:4321
npm run build     # Production build to ./dist/
npm run preview   # Preview production build locally
```
